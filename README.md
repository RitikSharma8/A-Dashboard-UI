# A-Dashboard-UI

A modern, responsive, and intuitive **Dashboard UI** built for a previous project. 

## Features
- Clean **midnight blue & orange accent** theme
- **Responsive design** for desktop and mobile
- **Stats cards** displaying sales, orders, and customers
- **Recent orders table** with vertical scrolling
- **Floating back-to-top button**


## Tech
- HTML5
- CSS3 (Flexbox & Grid)
